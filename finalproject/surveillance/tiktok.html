Tiktok

The Newest and most Intense Version

Tiktok works in a similar way to Google and Instagram. Every swipe, how long a user watches a video, what audio a user replays, the user's face, voice, location and even mirco-behaviors are being analyzed and collected by the platform. This data helps the system make better recommendations. It learns user's desires faster and better than the user themselves, predicts what the user wants to see and personalizes the content to each individual.

Tiktok's algorithm is buit around a hyper-personalized prediction loop. It modifies and shapes user behavior in real-time.

The platform uses behavior surplus to modify user's behavior, recommend hyper-personalized content and most importantly it makes a profit by selling information about user's next actions to advertisers.
